# SISTEMA CRUD MYSQL

Sistema completo de CRUD desenvolvido em Python com integração ao banco de dados MySQL via terminal.

##  Funcionalidades

- Cadastro de produtos
- Cadastro de vendedores
- Registro de vendas
- Atualização de informações
- Exclusão de registros
- Consultas no banco de dados
- Relatórios de vendas
- Integração com MySQL
- Sistema executado totalmente via terminal

---

##  Tecnologias Utilizadas

- Python
- MySQL
- PyMySQL
